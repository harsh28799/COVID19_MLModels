1. COVID-19 Contains Covid 19 data available with John Hoppkins University Portal or through Kaggle. 

2. LinearRegressionModel_COVID19 is the Linear Regression Model which can predict the number of cases which might arise with each passing days.
  The number of confirmed cases available with kaggle is used to design the model
  It also contains the model for India as well as for the world.
  
3. LinearRegressionByCountry is also a regression model as above but here the user can predict by the number of upcoming cases by country
